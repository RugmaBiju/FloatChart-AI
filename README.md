# FloatChat

## Overview

FloatChat is a conversational data exploration platform designed to simplify access to large-scale oceanographic datasets. It allows users to query, analyze, and visualize ARGO ocean data using natural language, eliminating the need for complex scripts or domain-specific tools.

The project focuses on improving accessibility and usability of scientific data for students, researchers, and non-expert users through an intuitive, chat-based interface.

---

## Problem Context

Oceanographic datasets generated by autonomous ocean floats are vast, multidimensional, and stored in scientific formats such as NetCDF. While this data is openly available, extracting meaningful insights typically requires programming knowledge and familiarity with specialized tools.

FloatChat addresses this gap by providing a natural language interface that translates user queries into structured data operations and visual outputs.

---

## Key Features

* Natural language querying of oceanographic data
* Processing and analysis of ARGO NetCDF datasets
* Interactive visualizations (graphs and charts)
* Chat-based user interface for seamless interaction
* Backend-driven computation to ensure data accuracy

---

## Technology Stack

* **Programming Language:** Python
* **Backend Framework:** FastAPI
* **Data Processing:** xarray, netCDF4, NumPy, Pandas
* **Visualization:** Matplotlib, Plotly
* **Natural Language Processing:** Large Language Models (LLMs)
* **Version Control:** Git, GitHub
